---
title: "Delete Flight Alert"
api: "DELETE https://api.aviation-ai.com/v1/alerts/{id}"
description: "Delete an existing flight alert"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique alert identifier to delete
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="deleted_alert" type="object">
  Information about the deleted alert
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      The deleted alert ID
    </ResponseField>
    <ResponseField name="flight_number" type="string">
      Flight number that was being monitored
    </ResponseField>
    <ResponseField name="alert_type" type="string">
      Type of alert that was deleted
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE "https://api.aviation-ai.com/v1/alerts/alert_789012" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.aviation-ai.com/v1/alerts/alert_789012', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const result = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.aviation-ai.com/v1/alerts/alert_789012',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
)

result = response.json()
```

</RequestExample>
