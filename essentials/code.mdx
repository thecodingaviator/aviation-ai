## Syntax Highlighting

Use triple backticks with the language name:

```python
def hello_world():
    print("Hello, world!")
```

```javascript
function helloWorld() {
    console.log("Hello, world!");
}
```

```typescript
interface User {
    name: string;
    age: number;
}

const user: User = {
    name: "John",
    age: 30
};
```
