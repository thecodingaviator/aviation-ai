The MDX syntax allows you to write JSX in your markdown content. You can import components, such as interactive charts, alerts, and more.

## Basic Syntax

### Headers

```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

### Text

**Bold text** with `**Bold text**`

_Italic text_ with `_Italic text_`

~~Strikethrough~~ with `~~Strikethrough~~`

### Lists

Numbered lists:
1. First item
2. Second item
3. Third item

Bullet lists:
- First item
- Second item
- Third item

### Links

[Link to Mintlify](https://mintlify.com)

### Code

Inline `code` with backticks

```javascript
// Code block
function hello() {
    console.log("Hello, world!");
}
```
