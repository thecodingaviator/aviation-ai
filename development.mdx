<Info>
**Prerequisite** You should have installed Node.js (version 18.10.0 or higher).
</Info>

Step 1. Go to your docs are located (where your `mint.json` file is) and run the following command:

```bash
mint dev
```

Step 2. Visit localhost:3000 to see your docs.

Step 3. Try making some changes!

Step 4. To publish your changes, just commit and push to your default branch.

## Custom Ports

Mintlify uses port 3000 by default. You can use the `--port` flag to customize the port Mintlify runs on.

```bash
mint dev --port 3333
```
